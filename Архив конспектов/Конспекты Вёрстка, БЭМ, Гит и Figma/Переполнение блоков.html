Переполнение блоков

Что происходит, если попробовать поместить блок размерами 500x500 в блок размером 300x600?

Вложим элемент div с классом inner (англ. «внутренний») в родительский div с классом outer (англ. «внешний»):

<div class="outer">
  <div class="inner"></div>
</div>

А теперь зададим в CSS блокам с классом inner ширину больше, чем у внешних блоков outer; установим тем и другим разную окраску:

.outer {
  width: 300px;
  height: 600px;
  background-color: rgb(1, 163, 114);
}

.inner {
  width: 500px;
  height: 500px;
  background-color: rgba(0, 120, 255, .8);
}

Результат:

Больший блок вышел за рамки меньшего, и это будет видно. Такая ситуация называется в CSS переполнением, а поведение родительского блока при переполнении можно определить свойством overflow (англ. overflow, «переполнение»). У него есть три возможных значения:

  • auto — значение по умолчанию,
  • scroll — скрывать зоны, которые выходят за границы блока, но показывать скролл (полосу прокрутки),
  • hidden — скрыть не поместившуюся часть целиком без скролла.

У свойства overflow есть подтипы с теми же значениями:

  • overflow-x — определяет поведение по горизонтали,
  • overflow-y — определяет поведение по вертикали.

Для примера добавим overflow со значением hidden:

.outer {
  width: 300px;
  height: 600px;
  background-color: rgb(1, 163, 114);
  overflow: hidden;
}

.inner {
  width: 500px;
  height: 500px;
  background-color: rgba(0, 120, 255, .8);
}
